# 🌴 Explorez Idjwi

Frontend du site **Explorez Idjwi**, une plateforme qui favorise le tourisme sur l’île d’Idjwi.  
Les utilisateurs peuvent découvrir les lieux à visiter, consulter les hébergements, lire les avis d’autres visiteurs et **effectuer des réservations**.  
Une **interface admin** permet de gérer le contenu du site (lieux, hébergements, avis, utilisateurs…).

**Site en production** : [Explorez Idjwi](https://tourisme-7cuz.vercel.app/)

---

## 🚀 Aperçu du projet

- Découverte des lieux touristiques
- Consultation des hébergements
- Lecture et publication d’avis
- **Réservation d’hébergements ou activités**
- Gestion admin du site

## ⚙️ Installation et lancement

### 1️⃣ Cloner le projet

```bash
git clone https://github.com/MapendoRomeo/tourisme
cd tourisme
```

### 2️⃣ Installer les dépendances

```bash
npm install
```

### 3️⃣ Lancer le serveur de développement

```bash
npm run dev
```

L'application sera disponible sur : `http://localhost:5173`
